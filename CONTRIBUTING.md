# Contributing

Thanks for contributing to Family Calendar.

## Development Setup

1. Fork and clone the repository.
2. Install dependencies.
3. Start the app in development mode.

```bash
npm install
npm run dev
```

App URL: http://localhost:3000

## Build Check

Before opening a PR, run:

```bash
npm run build
```

## Project Conventions

- TypeScript and Next.js App Router
- Keep changes focused and minimal
- Avoid unrelated refactors in feature/fix PRs
- Preserve existing API behavior unless intentionally changed

## Commit and PR Guidelines

- Use clear commit messages that describe what changed and why.
- One topic per PR.
- Include screenshots for UI changes.
- Include reproduction and validation steps for bug fixes.

## Documentation

If you change behavior, update docs in the same PR:

- `README.md` for user-facing behavior
- `docs/API.md` for API changes

## Reporting Bugs and Requesting Features

Use the GitHub issue templates.
